# Ruoping Gao Portfolio

Personal portfolio site for Ruoping Gao.

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- MDX for content
- Once UI
- Sass (SCSS)

## Development

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - start dev server
- `npm run build` - build for production
- `npm run start` - start production server
- `npm run lint` - lint codebase
